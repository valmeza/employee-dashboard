# Employee Dashboard

This dashboard keeps track of employees with the ability to add, edit and delete employees.

## Programming Languages

Java 11, 
Spring Boot 2.4.4,
Spring Framework,
Gradle,
JUnit5,
Mockito,
Spring Data JPA Hibernate,
Thymeleaf
